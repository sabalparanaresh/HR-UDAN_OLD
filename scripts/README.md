# Scripts Directory

This directory contains various utility scripts for maintaining, testing, and refactoring the HR-UDAN codebase.

- `extraction/`: Scripts for extracting data or schemas from the databases
- `migration/`: Scripts for migrating databases or codebases
- `refactoring/`: Scripts used in the transition and refactoring phases
- `testing/`: Scripts for running automated or ad-hoc tests
- `utils/`: Assorted file movement, structure generation, or fixing scripts
- `validation/`: Scripts for analyzing, validating, or checking codebase integrity
